# 日本語見開き縦書きPDFのOCR (Gemini API版)

Google Gemini APIを使用して、日本語の縦書き・見開きPDFを高精度にテキスト化するツールです。
国立国会図書館デジタルコレクションのような、古い書籍のスキャンデータ（見開き2ページ、縦書き、2段組など）の文字起こしに最適化されています。

## 特徴

*   **完全ローカル動作**: ブラウザ（HTMLファイル）だけで動作します。サーバーへのアップロードは不要です。
*   **Gemini API連携**: 最新のGemini 2.5 Flash / 2.0 Flash モデルを利用可能。
*   **見開き自動分割**: 見開き（2ページ分）の画像を自動で左右に分割して認識します。
*   **高精度プロンプト**: 専門的なプロンプトを内蔵しており、段落の保持やノイズ除去を強力に行います。
*   **レイアウト設定**: 「見開き（自動分割）」と「単ページ（分割なし）」を切り替え可能。

## 使い方

1.  **APIキーの取得**:
    *   [Google AI Studio](https://aistudio.google.com/app/apikey) でAPIキーを取得してください（無料枠で利用可能）。
2.  **ツールの起動**:
    *   `index.html` をブラウザ（Chrome推奨）で開いてください。
3.  **設定**:
    *   APIキーを入力し、「設定を保存する」を押します。
    *   モデルを選択します（通常は `Gemini 2.5 Flash` 推奨）。
    *   **レイアウト設定**を確認します。
        *   **見開き（自動分割）**: スキャン画像が本の見開き（左右2ページ）の場合。
        *   **単ページ（分割なし）**: 1ページずつスキャンされている場合。
4.  **実行**:
    *   PDFファイルを選択します。
    *   開始ページと終了ページを指定し、「OCR処理を開始する」を押します。
5.  **結果**:
    *   画面下部にテキストが出力されます。「全てコピーする」ボタンでクリップボードにコピーできます。

## 注意事項

*   APIキーはブラウザのローカルストレージ（お使いのPC内）にのみ保存され、外部サーバに送信されることはありません。
*   大量のページを一気に処理するとブラウザのメモリ不足になる可能性があります。100ページ程度ごとの処理をお勧めします。
